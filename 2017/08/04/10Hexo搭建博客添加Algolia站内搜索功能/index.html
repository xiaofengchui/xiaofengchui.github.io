注意

这里需要注意的是，每次新发布文章，即执行：
Hexo clean
Hexo g
Hexo d
之后需要执行一下：
Hexo clean
Hexo algolia
来更新Algolia索引，不然搜索不到你上次更新索引之后发布的文章。
